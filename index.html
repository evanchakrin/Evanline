How to Use

Modes

Camber

Quick mode still works for fast checks. For best repeatability, switch to Precision, use a rigid jig, capture a Level baseline around the car, then take repeated forward and reversed readings before saving camber.

Toe

Use a straight edge or known straight-ahead reference first. Precision mode keeps that reference inside a shared session so the same fixture, baseline, and reversal workflow can be reused left-to-right.

Level

Precision mode uses Level to capture FL, FR, RL, and RR baseline points. This turns one quick level check into a session-wide reference plane for later trust scoring.

Pitch

Pitch remains available, but the precision workflow prioritizes repeatable camber, toe, and baseline setup first. Use it to spot front/rear attitude changes inside the same session.

Step-by-Step

1
Start a precision session Switch from Quick to Precision workflow when you want the app to track a device reference, fixture profile, baseline points, and repeated capture sets.
2
Calibrate the hardware stack Capture a trusted device reference, save or select a named fixture profile, and prefer a reversible 3-point jig that lands on the same wheel datum every time.
3
Measure baseline points around the car Use Level mode to capture FL, FR, RL, and RR. The app uses those points to report whether your baseline plane looks trusted, approximate, or too noisy.
4
Capture forward and reversed sets For each side, take repeated settled readings in the normal fixture orientation, then flip the jig or phone in the reversible direction and capture a reversed set to expose mounting bias.
5
Save only when the report looks trustworthy Precision saves now include repeatability, reversal bias, baseline quality, capture count, and a verdict so you can decide whether the number is good enough for comparison or adjustment.

Important Notes

!
3D-printed jig expectations Aim for rigid 3-point wheel-face contact, avoid tire sidewalls, make the jig symmetric or reversible, and add hard phone registration so each re-seat lands in the same place.
!
Consumer-grade sensors still need real-world validation Phone sensors can be very repeatable with a disciplined workflow, but you should still benchmark your setup against a digital angle gauge or professional rack before trusting adjustment values.
!
Safari on iPhone required Motion sensors require Safari on iPhone (iOS 13+). You must tap “Enable Sensors & Start” and approve the permission prompt.
